# 📰 News App iOS

A modern and clean iOS News Application built using SwiftUI. The app fetches and displays the latest news articles with a beautiful Apple-style UI, dark mode support, and smooth user experience.

---

# ✨ Features

- 📱 Modern SwiftUI UI Design
- 🌙 Dark Mode Support
- 📰 Latest News Fetching
- 🖼 Async Image Loading
- 🔗 Read Full Article Support
- ⚡ Fast & Responsive UI
- 🎨 Apple-style Minimal Design
- 🚀 Launch Screen + Splash Screen
- 📐 Clean Component Architecture

---

# 🌐 API Used

This app uses the GNews API for fetching the latest news articles in real-time.

API Documentation:

https://docs.gnews.io/
